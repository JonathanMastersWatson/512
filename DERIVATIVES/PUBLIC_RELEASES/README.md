# Public Documents

This folder contains explanatory and contextual materials intended for
public distribution.

Documents in this folder:
- describe the purpose and implications of 512,
- provide high-level context and framing,
- and may be quoted or circulated externally.

Nothing in this folder defines, modifies, or overrides the 512 Kernel.

For authoritative constraints, rules, and definitions, refer to the
kernel and protocol documents in this repository.
